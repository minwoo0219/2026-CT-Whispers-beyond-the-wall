# 2026-CT-Whispers-beyond-the-wall
Whispers Beyond The Wall
왕으로부터의 탈출

당신의 선택에 따라 엔딩이 달라지는 스릴 공포게임
박하윤(팀장), 이민우, 임서하, 이주연, 정찬미

 


이 게임은 당신의 선택에 따라 결말이 달라지게 됩니다,  과연 당신은 한 번에 왕을 피해서 당신의 사랑인 윌리엄을 만날 수 있을까요?


엘리노어
당신이자, 이 게임의 여주입니다.

엘리노어는 사랑하는 사람 윌리엄이 있기에 정략결혼을 거부하며 왕실 사람들과 멀리합니다, 그러던 어느날 눈을 떴는데 낮선 공간에 갇혀있다는 것을 알고 탈출 하기 위하여 노력하는데...성의 비밀을 알아버리고 말았네요.



제프리
엘리노어 정략결혼 상대인 제프리, 엘리노어는 성에 갇혀 막막하던 찰나 제프리를 만나게 되는데... 자신에게 악감정이 없는 것 같다고 생각이 든다.  

"어쩌면 내가 탈출하는 것을 도와줄 수 도 있어."



왕
엘리노어가 가장 증오하는 사람이자, 엘리노어를 죽이려고 하는 이 나라의 왕

아마 제정신이 아닌 것 같다고 추정된다..
